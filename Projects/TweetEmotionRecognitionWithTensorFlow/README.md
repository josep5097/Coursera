# Guided Project: 
Create a recurrent neural network and train it on a Tweet Emotion dataset to learn to recogniza emotions in tweets.

## Dataset 
The dataset has thousands of tweets each classified in one of 6 emotions. This is a multi class classification problem in the Natural Language Processing Domain.

## Framework
TensorFlow

## Objectives
*   Using a Tokenizer in TensorFlow
*   Padding and truncating Sequences
*   Creating and Training a Recurrent Neural Network
*   Using NLP and Deep Learning to perform Text Classification

## Duration of Guided Project 
2-hour long
